Mission: Residential Energy Conservation — using machine learning to help homeowners reduce energy usage.

Problem: Predict residential building energy consumption based on size, occupancy, appliances, temperature, and day type.

Dataset: Energy Consumption Dataset by Govindaram Sriram (Kaggle), filtered to Residential buildings only.

Approach: Multivariate regression using SGDRegressor (gradient descent), Decision Tree, and Random Forest; best model saved as best_energy_model.pkl.
